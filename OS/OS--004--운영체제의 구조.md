# 운영체제의 구조
운영체제는 다음과 같은 작업을 관리한다.
#### 1. CPU 스케줄링
- CPU의 권한을 누구에게 넘겨줄지 조절한다.

#### 2. 메모리 관리
- 한정된 메모리를 효율적으로 사용하기 위해 이를 관리한다.

#### 3. 파일 관리
- 디스크에 파일을 어떻게 보관할지 관리한다.

#### 4. 입출력 관리
- 컴퓨터와 다른 입출력장치간 데이터를 주고 받는 것을 관리한다.

#### 5. 프로세스 관리
- 프로세스의 생성과 삭제 작업을 한다.
- 자원을 할당하고 반환한다.
- 프로세스 간 협력이 가능하게 한다.

#### 6. 기타
- 보호 시스템
- 네트워킹
- 명령어 해석기(command line interpreter)
